# ATM Project

## Türkçe
Java ve Spring Boot öğrenirken geliştirdiğim basit bir ATM projesi. Bu projede backend tarafını nasıl kuracağımı ve veritabanı işlemlerini yapmayı denedim.

### Özellikler
- Hesap oluşturma
- Para yatırma
- Para çekme
- Bakiye sorgulama

### Kullanılan Teknolojiler
- Java
- Spring Boot
- PostgreSQL (veya veritabanı ne kullandıysam onu ayarlayabilirim)

### Nasıl Çalıştırılır?
Projeyi bilgisayarınıza indirin ve bir IDE (IntelliJ veya Eclipse) ile açın. Veritabanı ayarlarını `application.properties` dosyasından yapıp başlatabilirsiniz.

---

## English
This is a simple ATM project I built to practice Java and Spring Boot. I wanted to learn how to set up the backend and handle database operations.

### Features
- Create account
- Deposit money
- Withdraw money
- Check balance

### Technologies
- Java
- Spring Boot
- PostgreSQL

### How to Run
Download the project and open it with an IDE like IntelliJ or Eclipse. Configure the database settings in `application.properties` and run the application.
